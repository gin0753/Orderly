# Orderly UI Blueprint v1

## Product Direction

Orderly is a production-grade restaurant ordering platform.

It combines a clean ecommerce visual style with a food-ordering workflow.

## UI References

- Shopify Dawn: clean ecommerce layout, product cards, cart patterns, spacious design
- Uber Eats: restaurant menu structure, category chips, mobile ordering flow, bottom cart bar

Orderly should not clone either product. It uses them as UI and UX references.

## Visual Direction

- Clean, modern, commercial-quality
- Light neutral background
- White cards
- Dark slate text
- Warm orange-red accent
- Rounded surfaces
- Soft shadows
- Mobile-first ordering experience

## Menu Page Structure

1. Site header
2. Restaurant hero
3. Category tabs
4. Menu sections
5. Product cards
6. Cart drawer later

## Stage 2 Components

- SiteHeader
- MenuHero
- CategoryTabs
- MenuBrowser
- MenuSection
- ProductCard
- EmptyMenuState

## Responsive Rules

Product grid:

- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3 columns

Mobile menu:

- Compact header
- Compact hero
- Horizontal category chips
- Product list/cards
- Bottom cart bar later

## Interaction Rules

Category tabs:

- All shows all categories
- Category tab filters menu sections
- Mobile tabs scroll horizontally

Product card:

- Hover raises card slightly
- Add to cart button visible in v1
- Product modal comes in v2

Cart:

- Header cart button exists in v1
- Cart drawer comes in v2
